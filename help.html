Help!

Lista de compras

Feita para simplificar o seu dia a dia, a lista de compras é de fácil uso, basta preencher o produto e quantidade e clicar no botão adicionar + .

Caso queira remover um item basta clicar no botão vermelho x para remover.

Lista de convidados

Mais simple ainda do que a lista de compras. Pois precisamos apenas de uma informação para montar sua lista, basta informar o nome do convidado e clicar no botão verde + para adicioná-lo.

Para remover basta clicar no botão vermelho x da linha desejada. Simple né ?!

Lista de estudos

Na lista de estudos temos uma mecânica um pouco diferente.

Para adicionar um item é assim como nas demais listas, muito simples, bastando clicar em adicionar + .

Porém ela conta com uma opção de marcar como concluido, bastando marcar como "check".

E caso queira remover o item basta clicar duas vezes sobre o mesmo.

Imprimir lista

Para salvar ou imprimir a sua lista basta clicar no botão Salvar/Imprimir, será aberto uma página no seu dispositivo onde pode escolher imprimir ou salvar como PDF.
